Positioning

<button class="btn fixed-top rounded">fixed-top</button>

This would make a button fixed to the top of the page. Useful for creating navbars and such elements.


<button class="btn fixed-btm rounded">fixed-btm</button>

This would make a button fixed to the bottom of the page. Useful for creating footers and such elements.


<button class="btn sticky rounded bg">sticky</button>

Note: Fixed must be used on a background, this is why "bg" is in the class above.

Creates a sticky button that follows the page when scrolled past. Useful for navbars and such elements.
